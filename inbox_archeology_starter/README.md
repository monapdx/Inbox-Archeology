## What this is

Inbox Archeology transforms your Google Takeout data into a searchable, visual archive.

- Fully local
- Privacy-first
- Designed for exploration, not just storage

---

## Quick Start

```bash
git clone https://github.com/monapdx/Inbox-Archeology.git
cd Inbox-Archeology
npm install
npm run dev
```
